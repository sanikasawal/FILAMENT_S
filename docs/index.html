How it works

No prompt. No button.
Just presence.

Filament runs silently in your browser, watching your screen and listening to your voice. When it spots something actionable, it reaches into your Gmail and Drive and speaks — before you even think to ask.

01
👁

Watches your screen

Captures one frame every 3 seconds via getDisplayMedia() and streams it to the Gemini Live API in real time.

02
🎙

Listens to your voice

Captures microphone audio as PCM Int16 at 16kHz via AudioWorklet and streams it bidirectionally — speak naturally, get voice back.

03
🔍

Searches your Workspace

When Gemini spots something worth acting on, it calls fetch_workspace_context — querying your Gmail and Drive via OAuth 2.0.

04
🔊

Speaks the answer

Synthesizes a short, relevant nudge and plays it back as audio — the specific name, date, or file you need, right when you need it.

Features

Everything connected,
nothing in your way.

🌅

Morning Brief

Open a new tab before 11am and Filament greets you with a spoken summary of your unread emails and recent files — no prompting needed.

⚡

Proactive Screen Analysis

Working on a spreadsheet with empty cells? Filament notices and searches your inbox for the data that should be there.

🔄

Navigation Intent Reader

Switch from a Google Doc to Gmail and Filament detects your intent — already searching for the right thread before you start typing.

💬

Voice Q&A

Ask anything: "What was the last email I sent?" or "Find the file about the API deadline." Filament searches and answers in voice.

🔒

Your Data, Your OAuth

No data stored. The OAuth token lives only in your browser session and is used exclusively for real-time API calls on your behalf.

☁️

Fully Cloud-Native

Backend deployed on Google Cloud Run with automated deployment scripts. Scales to zero when idle, up to 5 instances under load.

Tech Stack

Built on Google's
most capable APIs.

Every layer is purpose-built for real-time, multimodal AI — from the AudioWorklet PCM pipeline to ADK's LiveRequestQueue bridging WebSocket audio into Gemini's bidirectional stream.

Gemini Live API Google ADK Google GenAI SDK Gmail API Google Drive API Google Cloud Run Cloud Build FastAPI (Python 3.11) Chrome Extension MV3 WebSocket AudioWorklet Web Audio API OAuth 2.0

Built for the #GeminiLiveAgentChallenge

Filament was created for the purposes of entering the Build With AI NYC Hackathon 2026 — Live Agents category. It uses the Gemini Live API (gemini-2.5-flash-native-audio-latest), Google ADK, and is hosted entirely on Google Cloud Run.
